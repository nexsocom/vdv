# Van de Veire Powersports - concept website

Static concept website ready for GitHub Pages.

## Upload
1. Create a new GitHub repository.
2. Upload every file/folder in this package to the root of the repo.
3. Go to Settings → Pages → Deploy from branch → main → root.
4. Open the GitHub Pages URL.
